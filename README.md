# Netflix Content Analysis: Exploring Movie Trends of the 1990s

## Overview

Netflix has grown from a DVD rental service into one of the world's largest entertainment platforms.

This project explores Netflix's content catalog through exploratory data analysis to understand movie trends from the 1990s.

A production company specializing in nostalgic entertainment styles wanted to research movies from this decade to better understand content characteristics and patterns.

---

## Business Problem

The production company wanted insights into:

- Movie duration trends
- Genre distribution
- Release patterns
- Characteristics of 1990s movies

These insights could support creative research and content strategy.

---

## Objective

Analyze Netflix movie data to discover:

- Common movie durations
- Popular genres
- Release trends
- Content characteristics

---

## Dataset

Dataset:

netflix_data.csv

Features include:

- show_id
- type
- title
- director
- cast
- country
- date_added
- release_year
- duration
- description
- genre
